{% trans "Vorwort" %}

{% blocktrans trimmed %} In dieser Datenschutzerklärung informiert RPTU4You darüber, welche personenbezogenen Daten bei der Nutzung der Website verarbeitet werden und welche Cookies hierbei zum Einsatz kommen. {% endblocktrans %}

{% trans "Besuch der Website ohne Anmeldung" %}

{% blocktrans trimmed %} Beim Aufruf der Website setzt RPTU4You einen CSRF-Cookie, um Cross-Site-Request-Forgery-Angriffe zu verhindern und die Integrität der Anwendung sicherzustellen. Der Cookie wird als zufällig generierte Zeichenfolge gespeichert und ermöglicht keine Identifikation einzelner Nutzerinnen oder Nutzer. {% endblocktrans %}

{% blocktrans trimmed %} Filtereinstellungen, die unter "Latest News" gewählt und über "Filter anwenden" bestätigt werden, werden ausschließlich temporär im jeweils aktiven Browser gehalten und nach dem Verlassen der Seite gelöscht. {% endblocktrans %}

{% trans "Registrierung und Authentifizierung" %}

{% blocktrans trimmed %} Für die Registrierung sind ein Benutzername und ein Passwort anzugeben. Der Benutzername wird im Benutzerkonto gespeichert. Das Passwort wird ausschließlich in gehashter Form abgelegt; eine Wiederherstellung des ursprünglichen Passworts ist nicht möglich. Zusätzlich wird dem Konto eine interne Identifikationsnummer zugewiesen, die keine Rückschlüsse auf persönliche Daten zulässt. {% endblocktrans %}

{% blocktrans trimmed %} Nach erfolgreicher Anmeldung setzt RPTU4You einen Session-Cookie, um den Login-Zustand aufrechtzuerhalten. Dieser Cookie verfällt automatisch am ersten Tag des Folgemonats oder sobald Sie sich aktiv abmelden. {% endblocktrans %}

{% trans "Trusted Accounts" %}

{% blocktrans trimmed %} Für die Beantragung eines Trusted Accounts ist eine Begründung erforderlich. Diese Begründung wird zusammen mit dem zugehörigen Benutzerkonto gespeichert, um die Entscheidung der zuständigen Mitarbeitenden nachvollziehbar zu dokumentieren. Die Daten bleiben unabhängig vom Ergebnis des Antrags erhalten. {% endblocktrans %}

{% trans "Kalenderfunktionen" %}

{% blocktrans trimmed %} Nach der Anmeldung können Nutzerinnen und Nutzer eigene Termine anlegen. Die Termininhalte werden gespeichert, um sie im persönlichen Kalender sowie auf der 4YouPage anzuzeigen. {% endblocktrans %}

{% blocktrans trimmed %} Kalenderdateien, die über die Funktion "Kalender importieren" hochgeladen werden, werden in der Datenbank von RPTU4You abgelegt, damit sie dauerhaft in der Anwendung zur Verfügung stehen. {% endblocktrans %}

{% trans "Präferenzeinstellungen" %}

{% blocktrans trimmed %} Über die Präferenzverwaltung können individuelle Filter für News-Kategorien, Standorte, Zielgruppen und Quellen hinterlegt werden. Die Einstellungen werden dem Benutzerkonto zugeordnet und bleiben bis zu einer Änderung durch die Nutzerin oder den Nutzer bestehen. {% endblocktrans %}

{% blocktrans trimmed %} Die Filter können jederzeit über die 4YouPage oder den Menüpunkt "Präferenzen" angepasst werden. Änderungen wirken sich unmittelbar auf die dargestellten Inhalte aus. {% endblocktrans %}

{% trans "Datensicherheit und Weitergabe" %}

{% blocktrans trimmed %} RPTU4You trifft organisatorische und technische Maßnahmen, um gespeicherte Daten vor unberechtigtem Zugriff zu schützen. Eine Weitergabe personenbezogener Daten an Dritte findet nicht statt, es sei denn, gesetzliche Vorgaben verpflichten RPTU4You hierzu. {% endblocktrans %}

{% blocktrans trimmed %} Nutzerinnen und Nutzer können jederzeit Auskunft über die zu ihrer Person gespeicherten Daten verlangen und die Berichtigung oder Löschung veranlassen, sofern keine entgegenstehenden gesetzlichen Aufbewahrungspflichten bestehen. {% endblocktrans %}
